# Country Weather Flask App

A web application that searches for country information and displays live weather data for its capital city.

## Live Demo
https://country-weather-flask.onrender.com/

## Features
- Search any country by name
- Displays capital city and weather information
- Shows temperature, humidity, wind speed, and weather condition
- Handles invalid country names gracefully

## How to run locally
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python app.py`
3. Open `http://127.0.0.1:5000` in your browser

## Built with
- Python
- Flask
- requests
- BeautifulSoup
- Deployed on Render
